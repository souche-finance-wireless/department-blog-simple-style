## 文档工具基于Vuepress

```bash
# 安装 vuepress
yarn global add vuepress # OR npm install -g vuepress

# 创建一个 md 文件，试着写一些东西吧。
echo '# Hello, World.' > Hello.md

# 启动本地开发环境服务器
vuepress dev

# 构建静态文件
vuepress build
```
